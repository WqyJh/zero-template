# zero-template

go-zero template for faster startup.

## api

Install dependencies

```bash
cd api/
make init
```

Generate api

```bash
make api
```

Generate doc

```bash
make doc
```

Generate model

```bash
make model
```

Start/Stop/Restart api

```bash
../script/start.sh
../script/stop.sh
../script/restart.sh
```
